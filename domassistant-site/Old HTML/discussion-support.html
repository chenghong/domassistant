DOMAssistant Discussion

You can join the DOMAssistant Discussion Group to discuss features or get support for any potential problems you might encounter. There you can get updates through e-mail, an RSS feed or right there in the web site itself.

DOMAssistant Support

If you have encountered a bug with DOMAssistant, please report it at Issues. Please attach the code which isn't working, what web browser(s) it doesn't work in and what platform(s) you have tested.
