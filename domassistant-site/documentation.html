Documentation

DOMAssistant contains of 6 individual modules, where only the core DOMAssistant module is required. Any other module can be used without any relational dependency to each other. All modules are available as separate downloads, in the DOMAssistantCompressed.js file (for production) or in the DOMAssistantComplete.js file (for learning and development).

Within this Documentation section, you will find every available method documented together with some example code. The parameters they accept and their return value is specified in respective module page.
